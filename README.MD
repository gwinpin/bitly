# **Bitly shortener**

## **Getting Started**

**require:**
*apache mod_rewrite, curl, php curl.


## **Configuration file**

Configure file config/conf.php


```<?php
return [
  //connection DB
    'host' => 'localhost',
    'dbname' => 'dbname',
    'user' => 'user',
    'password' => 'password',
  //token from Bitly
    'token' => '' 
]; 
```
